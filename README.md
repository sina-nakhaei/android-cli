# android-cli

A powerful Kotlin-based Command Line Interface (CLI) designed to streamline Android development workflows.

## Features

- **Fast & Lightweight:** Built with Kotlin for rapid execution.
- **Easy Installation:** Up and running on Windows with a single PowerShell command.
- **Automated Updates:** The installer always fetches the latest stable release directly from GitHub.
